# Railsy: The Temporary Email Client
March 17th, 2024 (edited) / Mahdy Karam

[Railsy](https://crates.io/crates/railsy) is a CLI tool made to make generating and using temporary emails as easy as possible. I drew inspiration from the [Mailsy](https://github.com/BalliAsghar/Mailsy) project, but that was only available on homebrew and node.
