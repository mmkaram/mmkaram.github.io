Railsy

Get dispolsable emails right from your terminal!

December 11, 2024

Railsy is a CLI tool made to make generating and using temporary emails as easy as possible. I drew inspiration from the Mailsy project, but that was only available on homebrew and npm.

The code is here. Viewing incoming messages/message previews works, but other features are coming.
